# Shakespeare Text Indexer

## Overview
A Java-based full-text indexing engine that parses large documents and builds
a searchable word index using a custom binary search tree.

Each word is indexed with:
- total frequency count
- line numbers where it appears

## Features
- Custom binary search tree implementation
- Efficient keyword search
- Inorder traversal for sorted output
- Handles large text files

## How to Run
1. Ensure `shakespeare.txt` is in the project directory
2. Compile all `.java` files
3. Run the main indexing class
4. Enter a word to search and view its occurrences

## Tech Stack
- Java
- Binary Search Trees
- File I/O
